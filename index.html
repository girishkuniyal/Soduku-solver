Soduku-solver

It is a c++ source code for soduku matrix 6x6 solver which accepts digit from 1 to 6. it is a source code for built a program which is able to solve soduku matrix of 6x6 of Easy and Medium Level.the soduku puzzle is given by user and solution is given by our program.

How to Run and Contribute

Step for using solver (Windows user):

user will give a soduku for solving contains digit from 1 to 6. the empty space in soduku is represented by 0; example of 6x6 soduku problem is :

2   0   1   6   3   4
0   4   6   2   5   0   
1   2   5   0   6   3
4   6   3   5   0   2   
0   1   4   0   2   6
6   3   2   1   0   5

where 0 represents blank space in soduku.
